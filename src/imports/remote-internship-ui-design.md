Design a modern, clean, and consistent UI for a web application called “Remote Internship Management & Evaluation Platform”.

The platform has two user roles:

Admin (Employer)

Student (Intern)

🎨 Design Requirements

Style: Minimal, professional, modern SaaS dashboard

Theme: Light theme with soft neutral background (#F8FAFC or similar)

Primary color: Blue or Indigo for actions and highlights

Typography: Clean sans-serif (Inter / Poppins style)

Use consistent spacing (8px grid system)

Soft shadows, rounded corners (8px–12px radius)

Clean card-based layout

Sidebar navigation layout for dashboard pages

Responsive web layout (desktop-first)
